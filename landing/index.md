# Kintone・Slack・Google Sheets の手作業を小さく自動化します

日本国内の企業・kintone 支援会社向けに、kintone、Slack、Google Sheets、CSV まわりで毎週くり返している確認・転記・通知・整形作業を、小さな単位で改善します。

大きなシステム刷新ではなく、今ある運用を活かしながら、現場の手間とミスを少しずつ減らすための支援です。

## よくある課題

- kintone の入力漏れに後から気づく
- ステータス変更後の連絡が手作業になっている
- 期限超過の確認が人任せになっている
- CSV インポート前の整形に毎回時間がかかる
- Google Sheets と kintone の二重管理が増えている
- 小さな改善をしたいが、社内で手が回らない

## お手伝いできること

- kintone JavaScript カスタマイズ
- 保存前・ステータス変更前の入力チェック
- Slack 通知、リマインド
- CSV / Google Sheets 連携の下準備
- 小規模 API 連携の検証
- 業務フローの棚卸しと改善提案
- 既存カスタマイズの軽微な修正

## 自動化の例

- 「承認依頼」ステータスに進む前に必須項目をチェック
- 期限を過ぎたレコードに警告を表示
- ステータスが「対応完了」になったら Slack に通知
- CSV の空白、表記ゆれ、必須列をインポート前にチェック
- 週次報告に必要なデータを整理しやすくする

1. 初回ヒアリング

## 対応方針

無理に大きな開発にはしません。まずは、現場で効果が見えやすく、リスクを管理しやすい小さな改善から進めます。

初回相談では、API トークン、顧客データ、機密資料は不要です。

# Kintone Ops Automation Sprint

日本国内の企業・kintone 支援会社向けに、kintone、Slack、Google Sheets、CSV まわりの小さな手作業を、無理のない範囲で自動化します。

大規模なシステム刷新ではなく、現場で毎週・毎月くり返している確認、転記、通知、CSV 整形、進捗チェックを対象にした小規模支援です。

## 対象

- 日本国内で kintone をすでに使っている中小企業
- 日本国内で kintone 導入・伴走支援をしているパートナー企業
- 社内の手作業や確認漏れを少しずつ減らしたいチーム
- 大きな開発予算を取る前に、小さく改善効果を見たいチーム

## 対応できること

- kintone JavaScript カスタマイズ
- ステータスや入力値に応じた保存前チェック
- Slack 通知、リマインド、簡単なアラート
- CSV / Google Sheets 連携の下準備
- API 連携の小規模な検証
- 業務フローの棚卸しと改善ポイント整理
- 既存カスタマイズの軽微な修正、保守

## サービス例

### Workflow Audit

目安: 20,000円から

現在の業務フローをヒアリングし、kintone や周辺ツールで小さく改善できるポイントを整理します。

成果物:

- 改善候補リスト
- 優先順位
- 小さな実装案
- 注意点、リスク

### Mini Automation Sprint

目安: 80,000円から

1つの小さな業務課題に絞って、JavaScript カスタマイズ、Slack 通知、CSV 整形などを実装します。

対象例:

- 必須項目の未入力防止
- ステータス変更時の通知
- 期限超過の警告
- CSV インポート前の整形

### Monthly Care

目安: 20,000円/月から

小さな修正、設定確認、改善相談を継続的に行います。

## 進め方

1. 30分程度の初回相談
2. 課題と対象範囲の整理
3. 小さな実装案と見積り
4. デモ環境または検証環境で確認
5. 本番反映、または運用メモ作成

## 対応範囲の考え方

対応するのは、小さく切り出せる業務改善です。大規模な基幹システム導入、複雑な権限設計、全社横断の大規模移行は、単独対応ではなく専門チームやパートナー企業との分担を前提にします。

また、公式サイボウズパートナーであることを示すものではありません。必要に応じて、既存のパートナー企業様の補助・開発支援という形で対応します。

## 相談時に共有いただきたい情報

- 現在困っている作業
- 対象の kintone アプリ概要
- 使っている周辺ツール
- 毎月・毎週どれくらい時間がかかっているか
- できれば避けたい運用ミス

機密情報、API トークン、顧客データは初回相談では不要です。
